# JavaScript – Exemplos de Uso

Coleção de exemplos práticos em JavaScript focados nos principais conceitos da linguagem aplicados ao Front-End.

## Objetivo
Praticar JavaScript puro (Vanilla JS) por meio de exercícios e mini-projetos, explorando manipulação do DOM, eventos, armazenamento, requisições assíncronas e validações.

## Conteúdos Abordados
1. Manipulação do DOM  
2. Alteração dinâmica de estilos  
3. Armazenamento local (LocalStorage)  
4. Requisições assíncronas com Fetch  
5. Animação simples  
5A. Contador  
5B. Manipulação de imagens  
6. Lista de tarefas (To-Do List)  
7. Validação de formulários  
8. Tratamento de erros  
9. Async / Await  
10. Menu hambúrguer  
11. Carrossel  
11A. Contador (variação)  
11B. Contador (exercício prático)

## Tecnologias Utilizadas
- JavaScript (Vanilla)
- HTML5
- CSS3

## Escopo
Projetos focados exclusivamente em **lógica JavaScript e interação com a interface**, sem uso de frameworks ou bibliotecas externas.

## Aprendizados
- Manipulação do DOM e eventos
- Programação assíncrona
- Validação e tratamento de erros
- Organização de lógica em pequenos projetos
- Interatividade no Front-End
