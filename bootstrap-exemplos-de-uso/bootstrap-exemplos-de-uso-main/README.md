# Bootstrap – Exemplos de Uso

Repositório com **exemplos práticos e progressivos de Bootstrap**, cobrindo desde a instalação até a construção de layouts completos e projetos finais.

## Objetivo
Estudar e praticar os principais recursos do **Bootstrap**, entendendo como cada componente funciona na prática e como combiná-los em layouts reais.

## Estrutura dos Exemplos

### 0. Guia Completo Bootstrap
Introdução geral ao framework, conceitos e estrutura básica.

### 0A. Guia de Ícones Bootstrap
Uso dos Bootstrap Icons e integração com páginas HTML.

### 1. Instalação do Bootstrap
- **1A. CDN** – Uso via link externo  
- **1B. Arquivos locais** – Download e inclusão manual  
- **1C. NPM** – Instalação via gerenciador de pacotes  

### 2. Sistema de Grid
Criação de layouts responsivos utilizando o grid do Bootstrap.

### 3. Botões
Estilos, variações e estados de botões.

### 4. Formulários (Sub-CSS)
Estilização de formulários utilizando classes utilitárias.

### 5. Cards
Componentes de cards e organização de conteúdo.
- **5A. Grid com Cards**
- **5B. Formulário com Cards**

### 6. Navbar
Criação de menus de navegação responsivos.

### 7. Modal
Uso de janelas modais para interação com o usuário.

### 8. Carrossel
Criação de sliders e carrosséis de imagens/conteúdo.

### 9. Alerts & Progress (Sub-JS)
Alertas, barras de progresso e componentes dependentes de JavaScript.

### 10. Layout Completo
Construção de uma página completa utilizando vários componentes.
- **10A. Navegação com Modal**
- **10B. Grid aplicado ao layout**

### 11. Projeto Art
Projeto final aplicando os conceitos estudados ao longo dos exemplos.

## Tecnologias Utilizadas
- HTML5  
- CSS3  
- Bootstrap  

## Aprendizados
- Uso prático do Bootstrap
- Construção de layouts responsivos
- Aplicação de componentes CSS e JS
- Organização de interfaces rápidas e reutilizáveis
