# Python Mobile – Exemplos de Uso com Flet

Projeto com exemplos práticos de desenvolvimento de aplicações mobile e desktop utilizando a biblioteca Flet em Python, permitindo criar interfaces multiplataforma com código simples e produtivo.

## Objetivo
Explorar o uso do Flet para criação de aplicativos em Python que podem ser executados no navegador, desktop e dispositivos móveis (Android e iOS), focando em prototipação rápida e interfaces interativas.

## Tecnologias Utilizadas
- Python
- Flet
- Flutter SDK (para geração de APK)
- VS Code
