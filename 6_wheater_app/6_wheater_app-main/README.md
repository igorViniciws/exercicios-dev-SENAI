# Weather App – Previsão do Tempo (Flet)

Aplicação mobile/desktop desenvolvida em Python utilizando a biblioteca Flet, que consome a API OpenWeatherMap para exibir a previsão do tempo atual e dos próximos dias de forma simples e intuitiva.

## Objetivo
Praticar o desenvolvimento de aplicações multiplataforma com Python, integrando interface gráfica, consumo de API externa, programação assíncrona e controle de tema (claro/escuro).

## Funcionalidades
- Exibição do clima atual da cidade selecionada  
- Previsão dos próximos dias  
- Seleção de cidades brasileiras  
- Alternância entre modo claro e modo escuro  
- Ícones dinâmicos conforme a condição climática  
- Interface responsiva com layout mobile  

## Tecnologias Utilizadas
- Python
- Flet
- OpenWeatherMap API
- HTTPX (requisições assíncronas)
