# PHP – Exemplos Básicos

Este repositório reúne exemplos práticos em PHP voltados ao aprendizado dos fundamentos da linguagem, manipulação de formulários, controle de fluxo, autenticação de usuários e operações básicas com banco de dados (CRUD).

O foco é consolidar a base necessária para desenvolvimento web backend com PHP.

---

## Conteúdo

1. Hello World  
2. Operações com variáveis  
2b. Operações com variáveis (variações)  
3. Formulário básico  
4. Condicionais  
4b. Mensagem de boas-vindas  
5. Cadastro de usuário  
6. Login  
7. Arrays  
8. Validação de formulário  
9. Conexão com banco de dados  
10. Inserção de dados  
11. Listagem de dados  
12. Atualização de registros  
13. Exclusão de registros  
14. Upload de arquivos  
15a. Sistema básico  
15b. Área restrita  
15c. Logout  

---

## Tecnologias Utilizadas
- PHP
- HTML
- CSS (básico)
- MySQL

---

## Objetivo do Projeto
Praticar os principais conceitos do PHP aplicados ao desenvolvimento web, incluindo:
- lógica de programação
- manipulação de dados
- autenticação de usuários
- integração com banco de dados
- organização básica de sistemas web

---

## Como Executar
1. Instale um servidor local (XAMPP, WAMP ou Laragon)
2. Coloque o projeto na pasta `htdocs`
3. Inicie o Apache e o MySQL
4. Configure o banco de dados quando necessário
5. Acesse pelo navegador
