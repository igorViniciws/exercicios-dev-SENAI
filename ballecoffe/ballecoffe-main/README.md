# ballecoffe

Landing page desenvolvida em HTML, CSS e JavaScript com foco em layout, responsividade e organização visual, apresentando receitas de café e bebidas populares entre desenvolvedores.

## Funcionalidades
- Página inicial com destaque e navegação
- Sessão de receitas em cards
- Menu responsivo com ícone hamburguer
- Galeria de imagens
- Layout moderno e estilizado
- Interações simples com JavaScript

## Estrutura
- index.html
- style.css
- efeitos.js
- /assets (imagens, ícones e favicon)

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Observações
Projeto desenvolvido para prática de front-end, sem backend ou persistência de dados.
