# Minimalix

**Minimalix** é um site educativo e fictício desenvolvido no SENAI com foco em apresentar conceitos de desenvolvimento web, Front-End e a história da Internet.  
O projeto serve como estudo e prática de HTML5, CSS3 e JavaScript, aplicando boas práticas de estrutura, estilo e interatividade.

---

## Tecnologias Utilizadas

- **HTML5**: Estrutura do conteúdo da página
- **CSS3**: Estilização e animações
- **JavaScript**: Comportamento e interatividade

---

## Sobre o Projeto

O site está dividido em seções claras para aprendizado e apresentação:

### Home
A porta de entrada para o desenvolvimento web, destacando os fundamentos de HTML5 e CSS3.

### Desenvolvedor Front-End
Explica o papel do desenvolvedor Front-End:  
- Estrutura com HTML  
- Estilo visual com CSS, incluindo animações  
- Interatividade e comportamento com JavaScript  

Essa seção mostra a importância do desenvolvimento por camadas e como cada tecnologia atua de forma complementar.

### História da Internet
Uma seção educativa com um vídeo incorporado que explica a evolução da Internet e como chegamos ao desenvolvimento web moderno.  

[Assista ao vídeo](https://www.youtube.com/embed/jlkvzcG1UMk)
