# Exercícios de Desenvolvimento

Esta pasta contém exercícios práticos desenvolvidos durante minha formação em Desenvolvimento de Sistemas, com foco em lógica de programação, fundamentos das linguagens e resolução de problemas.

O objetivo destes exercícios é demonstrar minha base técnica, evolução ao longo do aprendizado e aplicação dos conceitos que hoje utilizo em projetos pessoais e acadêmicos.
