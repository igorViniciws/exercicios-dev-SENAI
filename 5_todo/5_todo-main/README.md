# ToDo App – Gerenciador de Tarefas (Flet + Python)

Aplicativo de gerenciamento de tarefas desenvolvido em Python com Flet, utilizando banco de dados para persistência dos dados e interface moderna em modo escuro.

## Objetivo
Desenvolver um aplicativo desktop/mobile completo aplicando conceitos de interface gráfica, orientação a objetos, CRUD, persistência em banco de dados e organização de código.

## Funcionalidades
- Cadastro de usuário (nome)
- Adição de tarefas
- Listagem de tarefas
- Marcar tarefas como concluídas ou pendentes
- Exclusão de tarefas
- Filtros por abas (Todas, Pendentes, Concluídas)
- Persistência de dados em banco de dados
- Interface em modo escuro
- Feedback visual com SnackBar

## Tecnologias Utilizadas
- Python
- Flet
- SQLite (via camada de acesso ao banco de dados)
- Programação Orientada a Objetos (POO)
