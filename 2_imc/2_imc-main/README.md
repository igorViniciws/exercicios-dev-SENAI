# Calculadora de IMC (Python + Flet)

Aplicação desenvolvida em Python utilizando a biblioteca Flet para calcular o Índice de Massa Corporal (IMC), com interface simples, responsiva e suporte a modo claro e escuro.

## Objetivo
Praticar o desenvolvimento de aplicações desktop/mobile com Python, aplicando validação de dados, lógica de cálculo, controle de tema e interface gráfica interativa.

## Funcionalidades
- Cálculo automático do IMC
- Classificação do resultado (abaixo do peso, normal, sobrepeso e obesidade)
- Validação de entradas numéricas
- Aceita altura em metros ou centímetros
- Alternância entre modo claro e modo escuro
- Interface amigável e responsiva

## Tecnologias Utilizadas
- Python
- Flet
- Programação orientada a eventos
