# SENAI – Página Institucional

Projeto Front-End focado na criação de uma **página institucional do SENAI**, apresentando informações sobre cursos técnicos, projetos, localização e contato.

## Objetivo do Projeto
Praticar a construção de páginas institucionais utilizando **HTML5 e CSS3**, com foco em estrutura, organização de conteúdo e layout informativo.

## Seções da Página
- Menu de navegação (Início, Sobre, Endereço, Contato)
- Apresentação dos cursos técnicos do SENAI
- Seção sobre projetos e alunos participantes
- Informações gerais (local, duração e desafios)
- Endereço completo e mapa de localização
- Rodapé institucional

## Tecnologias Utilizadas
- HTML5  
- CSS3  

## Escopo
Projeto exclusivamente **Front-End**, sem backend ou funcionalidades dinâmicas.  
O foco é **layout, semântica e apresentação de informações institucionais**.

## Aprendizados
- Estruturação de páginas institucionais
- Organização de conteúdo informativo
- Uso de seções semânticas em HTML
- Layout e hierarquia visual com CSS
