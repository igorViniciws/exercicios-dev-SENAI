# Agência de Viagens 

Projeto Front-End focado exclusivamente na construção da **página inicial (Home)** de uma agência de viagens fictícia.

## Objetivo do Projeto
Praticar a estruturação de uma página Home utilizando **HTML5 e CSS3**, trabalhando layout, seções visuais e organização de conteúdo.

## Funcionalidades da Home
- Menu de navegação (Início, Destinos, Pacotes, Contato)
- Seção de **destinos populares** com cidades internacionais e nacionais
- Área de **busca de destino** com campo de texto e data
- Seção de **pacotes em promoção**, com descrição e valores
- Rodapé com informações de contato

## Tecnologias Utilizadas
- HTML5  
- CSS3  

## Escopo
Este projeto **não possui backend, banco de dados ou sistema de buscas real**.  
O foco foi **apenas o layout e a experiência visual da página inicial**.

## Aprendizados
- Organização semântica do HTML
- Estruturação de seções comuns em sites comerciais
- Criação de layouts simples e responsivos
- Boas práticas iniciais de Front-End
