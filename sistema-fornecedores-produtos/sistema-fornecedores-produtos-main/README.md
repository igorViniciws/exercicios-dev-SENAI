# sistema-fornecedores-produtos

Sistema em PHP para gerenciamento de fornecedores e produtos, com autenticação de usuários e operações CRUD integradas ao banco de dados MySQL.

## Funcionalidades
- Login e logout
- Controle de sessão
- Cadastro de fornecedores
- Cadastro de produtos
- Listagem de produtos
- Conexão com banco de dados
- Sistema com páginas restritas

## Arquivos principais
- index.php
- login.php
- logout.php
- valida_sessao.php
- conexao.php
- cadastro_fornecedor.php
- cadastro_produto.php
- listagem_produtos.php
- bd.sql
- styles.css

## Tecnologias
- PHP
- MySQL
- HTML
- CSS

## Observações
Projeto desenvolvido para fins educacionais, com foco em lógica de programação, autenticação e integração com banco de dados.
