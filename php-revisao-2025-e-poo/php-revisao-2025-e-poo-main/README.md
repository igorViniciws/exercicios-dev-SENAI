# PHP – Revisão e Programação Orientada a Objetos (POO)

Conjunto de exercícios e exemplos em PHP voltados à revisão da linguagem e à introdução dos conceitos de Programação Orientada a Objetos.

## Objetivo
Reforçar fundamentos do PHP e comparar a abordagem estruturada com POO, por meio de exercícios práticos desenvolvidos em ambiente educacional.

## Estrutura das Pastas
- **Revisão PHP – Alunos**  
  Exercícios de revisão dos conceitos básicos da linguagem PHP.

- **Revisão PHP – Professor**  
  Versões de referência dos exercícios para estudo e comparação.

- **POO vs Estruturada – Alunos**  
  Comparação prática entre programação estruturada e orientada a objetos em PHP.

- **Pilares da POO – Alunos**  
  Exercícios focados nos pilares da POO: encapsulamento, herança, polimorfismo e abstração.

- **Mini Exercício – Alunos**  
  Atividades práticas para fixação dos conceitos estudados.

- **Perfil Simples – Alunos**  
  Pequeno projeto aplicando POO em um sistema simples de perfil.

## Tecnologias Utilizadas
- PHP
- HTML (apoio)

## Aprendizados
- Revisão da sintaxe e lógica em PHP
- Diferenças entre código estruturado e orientado a objetos
- Aplicação prática dos pilares da POO
- Organização de código e reutilização
