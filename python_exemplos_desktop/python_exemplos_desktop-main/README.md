# Python – Exemplos Desktop (Interfaces Gráficas)

Conjunto de exemplos e exercícios em Python focados no desenvolvimento de aplicações desktop com interface gráfica, explorando diferentes componentes, layouts e integrações.

## Objetivo
Praticar a criação de interfaces gráficas em Python, desde telas simples até aplicações com lógica, validações e integração com banco de dados.

## Estrutura da Pasta
- **py_interface**  
  Interface gráfica básica para introdução aos conceitos de GUI.

- **py_interfaceradio**  
  Exemplo de interface utilizando botões de seleção (RadioButton).

- **ttkbootstrap**  
  Interface estilizada utilizando a biblioteca ttkbootstrap.

- **py_interface_trigonometria**  
  Aplicação desktop para cálculos trigonométricos.

- **py_calculadora**  
  Calculadora desktop com interface gráfica.

- **py_senha**  
  Interface para criação e validação de senha.

- **py_bd**  
  Interface gráfica integrada a banco de dados.

- **py_cadastro**  
  Sistema simples de cadastro com interface desktop.

- **desafio**  
  Desafio final aplicando os conceitos aprendidos ao longo dos exercícios.

## Tecnologias Utilizadas
- Python
- Tkinter
- ttkbootstrap
- Banco de dados (quando aplicável)

## Aprendizados
- Criação de interfaces desktop em Python  
- Uso de widgets, eventos e layouts  
- Estilização de interfaces  
- Integração entre interface, lógica e dados
